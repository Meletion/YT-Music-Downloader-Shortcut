# Youtube Music Downloader Shortcut, or YMDS.
This userscript + golang app combo let you download music by pressing Shift + D on your keyboard, when on YT music, or YT.

## Requirements:
- YT-DLP
- Golang, if you are building it from source

## How to use
1. Drop the built golang app into your music folder, or any other folder.
2. Install the included userscript from the repo
3. Go to youtube, click Shift + D
Congrats! You now have that song inside of the folder you put the golang app. Enjoy!

## How to build
1. Download the provided YMDS.go
2. Run go build . in the same folder
3. You now have it built for your OS


## Why?
I built this for myself, and couldn't find somewhere where I can just click a keybind and download a song.
